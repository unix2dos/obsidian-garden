# 1 Option

## 1.1 全局

- option + p ：粘贴板
- option + a ：截图后直接翻译
- option + s ：OCR 截图
- option + d ：选中文本打开翻译

## 1.2 软件

- option + e ：obsidian 打开今天
- option + w ：
	- chrome： 翻译（chrome 设置）
	- obsidian: 全部关闭
- option + ~ : Goland 和 cursor 互跳
